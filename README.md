# Python Math Toolkit

## About
A collection of beginner Python programs including calculators, unit converters, and simple equation solvers. Built while learning programming, algebra, and problem-solving.

———

## Programs included
- Basic calculator (addition, subtraction, multiplication, division)
- Unit converter (temperature, length, weight)
- Simple equation solver

———

## What I learned
- How to structure a repository
- How programming actually works; breaking a problem into steps, telling the computer what to do. 
- Variables (storing information). Variables are named storage for data, ex: num1 and num2 stored the numbers.
- Input and output. input() = user data collection. print() = results showing. Programs interact with users through input/output loops.
- Conditional logic. if = checking a condition. elif = checks next option if previous “if” failed. else = handling everything invalid.
- Python understands and uses mathematical symbols.
- Error handling, I faced a few challenges, just with formatting the indents.
- Loops. while True = repeat forever. break = stop loop. continue = restart loop early.
- Code structure and indentation. Indentation controls where code belongs. when True just be at the top with every other string indented below. Putting a colon (:) at the end of a line of code groups the rest that are indented underneath of it.
- Debugging skills. I noticed error messages and attempted to fix the code until it worked properly.
